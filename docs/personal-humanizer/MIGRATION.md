# Migration to Personal Humanizer 3.6.0

Replace the existing skill directory with this `personal-humanizer` folder or use the bundled runtime shipped with Vouch 0.19.3 RC.

Reply-mode changes:

- 4–35 words instead of 6–18;
- one sentence by default, two when necessary;
- lowercase is optional;
- plain observation and direct response are first-class candidates;
- humor and punchlines are optional;
- selection prefers the least constructed supported candidate;
- recent syntax and quote-card reuse are inspected structurally.

The standalone-post and long-form policies retain the 3.5.0 behavior.
