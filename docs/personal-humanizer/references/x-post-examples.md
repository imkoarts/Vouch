# Standalone X-Post Examples

These examples show functions, not phrase substitutions.

## Polymarket combos regression

Before:

> A trading tournament around Polymarket's new combos product gives the format a narrow test: how people use it when there is a reason to trade it actively. If combos extend beyond sports, the practical question is not only whether more categories can be listed. It is whether the combination remains legible enough that a trader can understand what is being priced.

Problems:

- abstract test framing
- vague `people`
- staged practical question
- `not only X; it is Y` reveal
- abstract noun density
- product-memo voice
- setup → reframing → thesis arc

Better:

> The tournament should show whether traders can quickly understand combo markets once volume picks up. If Polymarket expands them beyond sports, each market still needs to make clear what the price includes.

## Abstract test framing

Before:

> The launch gives the product a real-world test of user demand.

After:

> The launch should show whether customers will pay for it.

Keep `test` when actual criteria exist:

> The beta tests whether checkout works under 10,000 concurrent sessions.

## Staged practical question

Before:

> The practical question is not whether the feature can scale. It is whether users can still understand it.

After:

> Scaling the feature will not help if users cannot understand it.

Use only when supported.

## Abstract noun density

Before:

> Expansion increases the importance of combination legibility.

After:

> As Polymarket adds categories, traders still need to see what each combo includes.

## Product-memo voice

Before:

> The format creates an opportunity to test category expansion under active trading conditions.

After:

> The tournament should show how combos behave when trading picks up.

## Valid analytical post

Keep:

> Combo prices get harder to read as more legs are added. The tournament should show where traders start dropping off.

## Valid technical abstraction

Keep:

> Liquidity fragmentation raises slippage across smaller markets.

The terminology names a concrete mechanism.

## Valid question

Keep:

> Does Polymarket show the implied probability for each leg before a combo is placed?
