# Personal Humanizer Version

Version: 3.6.0

Personal Humanizer 3.6.0 changes reply-mode planning and selection. Replies now support plain observations, direct responses, qualifications, genuine questions, dry humor, and contextual extensions. The normal range is 4–35 words, one sentence by default and two when necessary. Lowercase and humor are optional. Selection prefers the least constructed supported candidate and checks recent structural reuse.

Standalone-post and long-form contracts retain the 3.5.0 evidence, paragraph-job, and anti-packaging protections.
